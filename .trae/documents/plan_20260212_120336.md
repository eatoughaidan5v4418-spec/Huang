## 创建可部署的网页计划

### 1. 设计基础网页内容
- 创建一个简洁美观的个人主页模板
- 包含：标题、简介区域、链接按钮、页脚
- 响应式设计，适配手机和电脑

### 2. 配置 GitHub Pages 部署
- 安装 `gh-pages` 部署工具
- 修改 `vite.config.js` 配置基础路径
- 在 `package.json` 添加部署脚本

### 3. 部署步骤说明
- 运行 `npm run deploy` 自动构建并发布
- 访问 `https://你的用户名.github.io/仓库名/` 查看网页

### 你需要准备
- 一个 GitHub 账号
- 在 GitHub 上创建一个新仓库（或使用现有仓库）
